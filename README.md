# Toqen

1. Start Anvil

```bash
anvil
```

2. Deploy Toqen

```bash
git clone https://github.com/toqenfactory/contracts.git
cd contracts
forge script script/Toqen.s.sol --fork-url http://localhost:8545 --broadcast
```

3. Components

```bash
git clone https://github.com/toqenfactory/components.git
cd components
npm run dev
```
